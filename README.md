## Responsive Food Website

Foodie is a modern and responsive food website built using HTML, CSS, and JavaScript. It showcases healthy food collections, menu items, customer reviews, and an online ordering section with a clean and user-friendly interface.

## Features

* Responsive Design
* Healthy Food Showcase
* Daily Deals Section
* Food Menu Display
* Customer Reviews
* Order Form
* Newsletter Subscription
* Social Media Links

## Technologies used

* HTML5
* CSS3
* JavaScript
